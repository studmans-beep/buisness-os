# Product Principles

## 1. Save Time

Every feature must save users time.

## 2. Create Value

The platform must help customers make or save more money than it costs.

## 3. Be Proactive

Don't wait for questions.

Identify problems and opportunities automatically.

## 4. Investigate Before Answering

Never guess.

Always check connected systems.

## 5. One Workspace

Users should not need to know where information is stored.

## 6. Beautiful by Default

Every screen should feel premium.

Simple.
Fast.
Professional.

## 7. Human First

The platform assists people.

Humans approve important decisions.

## 8. Learn the Business

Remember customer preferences.

Remember planner decisions.

Improve over time.

## 9. Make Every Planner Better

Help new staff perform like experienced staff.

Reduce dependency on individual knowledge.

## 10. Every Notification Must Help

Never show a problem without suggesting a solution.

Whenever possible:

- Explain the issue.
- Explain the impact.
- Recommend an action.
- Prepare the work.

## 11. Measure Value

The customer should regularly see:

- Hours saved.
- Miles saved.
- Fuel saved.
- Revenue opportunities found.
- Customer updates automated.

The value of the platform should always be visible.
