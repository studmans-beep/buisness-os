# Minimum Viable Product (MVP)

## Goal

Build an AI-powered operations platform that can read business emails, investigate company systems, and prepare work for approval.

The MVP will be tested in a haulage company before expanding to other industries.

---

# Module 1 - Email Intelligence

## Features

- Read Outlook emails
- Detect customer intent
- Categorise emails automatically
- Draft professional replies
- Flag urgent emails
- Search previous conversations

---

# Module 2 - Delivery Intelligence

## Features

- Check delivery status
- Check delays
- Retrieve PODs
- Find driver information
- Draft customer updates

---

# Module 3 - Quote Intelligence

## Features

- Read quote requests
- Extract pickup and delivery details
- Suggest pricing
- Draft quotations
- Send for approval

---

# Dashboard

The dashboard must show:

- New emails
- Delays
- Quote requests
- Tasks awaiting approval
- Recent activity

---

# AI Principles

The AI must:

- Investigate before answering
- Never guess
- Explain where information came from
- Complete work whenever possible
- Ask for approval before sending emails

---

# Success Criteria

The MVP is successful when a transport planner can:

- Open the platform
- Read a customer email
- Ask the AI for help
- Receive an accurate draft reply
- Approve and send the email

without manually searching multiple systems.
